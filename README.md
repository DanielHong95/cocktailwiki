## Installation

$ git clone https://github.com/DanielHong95/cocktailwiki.git

### Frontend Installation

$ cd cocktail-wiki/frontend
$ npm install

### Backend Installation

- $ cd cocktail-wiki/server
- $ npm install

## Running the Project

### Running the Frontend

- $ cd cocktail-wiki/frontend
- $ npm start

### Running the Backend

- $ cd cocktail-wiki/server
- $ npm install -g nodemon
- $ nodemon start
