## Installation

\$ git clone https://github.com/DanielHong95/cocktailwiki.git

### Frontend Installation

$ cd cocktail-wiki/frontend<br>
$ npm install

### Backend Installation

$ cd cocktail-wiki/server<br>
$ npm install

## Running the Project

### Running the Frontend

$ cd cocktail-wiki/frontend<br>
$ npm start

### Running the Backend

$ cd cocktail-wiki/server<br>
$ npm install -g nodemon<br>
$ nodemon start
